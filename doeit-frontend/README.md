# DOEIT – Personal Finance App

A Next.js frontend for the DOEIT personal finance application.

## Quick Start

```bash
cd doeit
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

**Demo login:** username `gabriel` / password `password`

## Pages

| Route         | Description                        |
|---------------|------------------------------------|
| `/login`      | Sign in screen                     |
| `/dashboard`  | Overview: balance, cash flow, allocations |
| `/cashflow`   | Transaction list + add new entry   |
| `/allocation` | Budget categories grid             |
| `/profile`    | User profile + financial summary   |
